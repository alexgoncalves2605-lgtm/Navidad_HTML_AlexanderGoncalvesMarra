En las primeras versiones de HTML se comenzaron a introducir elementos con la finalidad de mejorar la "presentación" del "contenido" de la página. Así aparecieron elementos para definir las fuentes, tamaños, colores, alineamiento de textos y bloques etc. En este caso utilizamos la función background para cambiar el fondo.

En el siguiente ejemplo vamos a utilizar el span.

Esta parte subrayada forma parte del SPAN. El span nos permite marcar o agrupar pequeños fragmentos de texto o elementos en línea para aplicarles estilos o scripts sin romper el flujo del texto.

En el siguiente ejemplo utilizamos el div.

El DIV es un contenedor de bloque que se usa para agrupar y organizar secciones de contenido en una página web, permitiendo aplicar estilos o scripts a todo ese bloque.

A diferencia del SPAN, el DIV es un contenedor de bloque que organiza secciones grandes de contenido, mientras que el SPAN es un contenedor en línea que marca pequeñas partes de texto sin romper el párrafo.
